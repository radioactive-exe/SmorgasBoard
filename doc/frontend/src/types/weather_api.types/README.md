# frontend/src/types/weather_api.types

This file contains all the types for responses from the Weather API, and
thus, from the backend of Smorgasboard.

## Author

Radioactive.exe
[GitHub Profile](https://github.com/radioactive-exe)

## See

[WeatherAPI](https://www.weatherapi.com/docs/)

## Interfaces

| Interface                                          | Description                                                       |
| -------------------------------------------------- | ----------------------------------------------------------------- |
| [Astro](interfaces/Astro.md)                       | Astrology information in the Weather API response.                |
| [Condition](interfaces/Condition.md)               | The shape of the Condition object inside Weather API responses.   |
| [Current](interfaces/Current.md)                   | The shape of the Current weather information for a location.      |
| [Day](interfaces/Day.md)                           | This includes information for the Day as a whole in responses.    |
| [Forecast](interfaces/Forecast.md)                 | The complete forecast response from the API.                      |
| [ForecastDay](interfaces/ForecastDay.md)           | The complete object for each forecast day requested from the API. |
| [Hour](interfaces/Hour.md)                         | Hourly weather information.                                       |
| [Location](interfaces/Location.md)                 | The general Location object inside Weather API responses.         |
| [LocationCurrent](interfaces/LocationCurrent.md)   | The complete current response from the API.                       |
| [LocationForecast](interfaces/LocationForecast.md) | The complete Forecast response from the API.                      |
