# frontend/src/functions/accessors

This file contains all the functions that relate to getting properties and
attributes.

## Remarks

This involves direct property accessing, extracting from strings, and
normalising any obtained values numerically.

## Functions

| Function                                                              | Description                                                              |
| --------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| [cssProperty](functions/cssProperty.md)                               | Gets the normal un-altered string value of a CSS property of an element. |
| [cssPropertyValue](functions/cssPropertyValue.md)                     | Gets the numerical value of the CSS property of an element directly.     |
| [normalisedCssPropertyValue](functions/normalisedCssPropertyValue.md) | Gets and normalises the CSS property of an element.                      |
