# frontend/src/constants

A file containing useful constants and literals to be used throughout the
frontend of SmorgasBoard.

## Author

Radioactive.exe
[GitHub Profile](https://github.com/radioactive-exe)

## Variables

| Variable                                          | Description                                                                                            |
| ------------------------------------------------- | ------------------------------------------------------------------------------------------------------ |
| [FULL_DAY_IN_MS](variables/FULL_DAY_IN_MS.md)     | The number of milliseconds per 24 hours. Used for the Weather panel's astrology and time calculations. |
| [MAX_FILE_SIZE](variables/MAX_FILE_SIZE.md)       | The Maximum accepted size when uploading a file in the Image upload input.                             |
| [MONTHS](variables/MONTHS.md)                     | Months of the year in an array that indexes them by JS/TS Date integers for months.                    |
| [VALID_FILE_TYPES](variables/VALID_FILE_TYPES.md) | Valid file types to be used in the Image uploading input for validation.                               |
| [WEEKDAYS](variables/WEEKDAYS.md)                 | Days of the week in an array that indexes them by JS/TS Date integers for days .                       |
