# frontend/src/types/weather_api.types

This file contains all the types for responses from the Weather API.

## Author

Radioactive.exe
[GitHub Profile](https://github.com/radioactive-exe)

## Interfaces

| Interface                                                                                   | Description                                                       |
| ------------------------------------------------------------------------------------------- | ----------------------------------------------------------------- |
| [Astro](../wiki/frontend.src.types.weather_api.types.Interface.Astro)                       | Astrology information in the Weather API response.                |
| [Condition](../wiki/frontend.src.types.weather_api.types.Interface.Condition)               | The shape of the Condition object inside Weather API responses.   |
| [Current](../wiki/frontend.src.types.weather_api.types.Interface.Current)                   | The shape of the Current weather information for a location.      |
| [Day](../wiki/frontend.src.types.weather_api.types.Interface.Day)                           | This includes information for the Day as a whole in responses.    |
| [Forecast](../wiki/frontend.src.types.weather_api.types.Interface.Forecast)                 | The complete forecast response from the API.                      |
| [ForecastDay](../wiki/frontend.src.types.weather_api.types.Interface.ForecastDay)           | The complete object for each forecast day requested from the API. |
| [Hour](../wiki/frontend.src.types.weather_api.types.Interface.Hour)                         | Hourly weather information.                                       |
| [Location](../wiki/frontend.src.types.weather_api.types.Interface.Location)                 | The general Location object inside Weather API responses.         |
| [LocationCurrent](../wiki/frontend.src.types.weather_api.types.Interface.LocationCurrent)   | The complete current response from the API.                       |
| [LocationForecast](../wiki/frontend.src.types.weather_api.types.Interface.LocationForecast) | The complete Forecast response from the API.                      |
