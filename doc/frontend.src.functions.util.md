# frontend/src/functions/util

This file contains miscellaneous utility functions.

## Remarks

Most of these functions are boolean checks, with a few class/DOM utility
functions in between.

## Author

Radioactive.exe
[GitHub Profile](https://github.com/radioactive-exe)

## Functions

| Function                                                                                              | Description                                                                     |
| ----------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------- |
| [areaCollisionWithElement](../wiki/frontend.src.functions.util.Function.areaCollisionWithElement)     | Checks if an inputted Area collides with a particular HTMLElement.              |
| [collidesWithAnyPanel](../wiki/frontend.src.functions.util.Function.collidesWithAnyPanel)             | Checks whether an inputted area collides with any panel at all.                 |
| [deleteAfterTransition](../wiki/frontend.src.functions.util.Function.deleteAfterTransition)           | Deletes/removes an element from the DOM after its transition duration.          |
| [getOptionLabelFromList](../wiki/frontend.src.functions.util.Function.getOptionLabelFromList)         | Gets the corresponding label for the option with the inputted value.            |
| [isValidOption](../wiki/frontend.src.functions.util.Function.isValidOption)                           | Checks whether a particular config value is valid/part of the possible options. |
| [removeClassAfterTransition](../wiki/frontend.src.functions.util.Function.removeClassAfterTransition) | Removes a specified class from an element after its transition duration.        |
| [wouldFit](../wiki/frontend.src.functions.util.Function.wouldFit)                                     | Checks if the given panels would fit in the potential layout dimensions.        |
