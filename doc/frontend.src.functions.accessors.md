# frontend/src/functions/accessors

This file contains all the functions that relate to getting properties and
attributes.

## Remarks

This involves direct property accessing, extracting from strings, and
normalising any obtained values numerically.

## Author

Radioactive.exe
[GitHub Profile](https://github.com/radioactive-exe)

## Functions

| Function                                                                                                   | Description                                                              |
| ---------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| [cssProperty](../wiki/frontend.src.functions.accessors.Function.cssProperty)                               | Gets the normal un-altered string value of a CSS property of an element. |
| [cssPropertyValue](../wiki/frontend.src.functions.accessors.Function.cssPropertyValue)                     | Gets the numerical value of the CSS property of an element directly.     |
| [normalisedCssPropertyValue](../wiki/frontend.src.functions.accessors.Function.normalisedCssPropertyValue) | Gets and normalises the CSS property of an element.                      |
| [normalisedValue](../wiki/frontend.src.functions.accessors.Function.normalisedValue)                       | Extracts and returns the normalised value of a string.                   |
| [numericalValue](../wiki/frontend.src.functions.accessors.Function.numericalValue)                         | Extracts the numerical value/content of a string using Regex.            |
