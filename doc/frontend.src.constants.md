# frontend/src/constants

A file containing useful constants and literals to be used throughout the
frontend of SmorgasBoard.

## Author

Radioactive.exe
[GitHub Profile](https://github.com/radioactive-exe)

## Variables

| Variable                                                                     | Description                                                                         |
| ---------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| [MAX_FILE_SIZE](../wiki/frontend.src.constants.Variable.MAX_FILE_SIZE)       | The Maximum accepted size when uploading a file in the Image upload input.          |
| [MONTHS](../wiki/frontend.src.constants.Variable.MONTHS)                     | Months of the year in an array that indexes them by JS/TS Date integers for months. |
| [VALID_FILE_TYPES](../wiki/frontend.src.constants.Variable.VALID_FILE_TYPES) | Valid file types to be used in the Image uploading input for validation.            |
| [WEEKDAYS](../wiki/frontend.src.constants.Variable.WEEKDAYS)                 | Days of the week in an array that indexes them by JS/TS Date integers for days .    |
